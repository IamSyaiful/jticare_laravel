# JTICare - Donation Platform with Midtrans

JTICare adalah platform donasi digital berbasis web yang bertujuan memfasilitasi penggalangan dana untuk komunitas terdampak bencana secara transparan dan aman.

### 🚀 Fitur Utama
- **Payment Gateway Integration:** Pembayaran otomatis menggunakan **Midtrans** (Gopay, Bank Transfer, dll).
- **Campaign Management:** Pengelolaan program donasi secara dinamis.
- **User Verification:** Sistem autentikasi aman untuk donatur dan admin.
- **Real-time Reporting:** Dashboard monitoring dana yang terkumpul.

### 🛠️ Tech Stack
- **Framework:** Laravel 11
- **Database:** MySQL
- **Integration:** Midtrans API
- **UI:** TailwindCSS 3
